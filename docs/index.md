## Quick Example

```yaml title="bear.config.yml"
name: my-platform

use:
  languages: [go, node]
  targets: [docker, cloudrun]
```

```yaml title="services/api/bear.artifact.yml"
name: api
target: cloudrun
depends:
  - shared-lib
env:
  PROJECT: my-gcp-project
```

```bash
# See what would happen
bear plan

# Execute the plan
bear apply
```
